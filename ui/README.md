# TrackEx UI

Personal Expense Tracking and Analytics Platform - Frontend

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The app will open at `http://localhost:3000`

## Available Scripts

- `npm start` - Run the app in development mode
- `npm build` - Build the app for production
- `npm test` - Run tests

## Features

- **Dashboard** - View summary of expenses and recent transactions
- **Add Expense** - Record new expenses with category, amount, and date
- **Analytics** - Visualize spending patterns with charts and graphs

## Project Structure

```
src/
├── components/        # Reusable UI components
├── pages/            # Page components
├── services/         # API service
├── styles/           # CSS files
├── App.js            # Main app component
├── index.js          # React entry point
```

## Technologies

- React 18
- React Router
- Recharts (for data visualization)
- Axios (for API calls)
